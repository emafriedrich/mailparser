MailParser
==========

MailParser is a parser for mail message.

Installation
------------

    % gem install mailparser

Usage
-----

    require 'mailparser'
    f = File.open('hoge.eml')
    m = MailParser::Message.new(f, :output_charset=>'utf-8')
    m.subject  #=> String
    m.body     #=> String
    m.part     #=> Array of Mailparser::Message
